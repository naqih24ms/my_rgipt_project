# Sustainable Investment Behavior Study

## Overview
This repository presents an empirical investigation into factors influencing environmentally responsible financial decision-making. The study focuses on how information signals and individual awareness shape investment-related judgments.

## Methodological Approach
- Structural Equation Modeling (variance-based approach)
- Machine learning-based predictive analysis
- Primary survey data collected from diverse respondents

## Research Focus
The study examines interactions between:
- Information reliability and perception
- Cognitive awareness of environmental issues
- Behavioral intention in financial contexts

## Repository Contents
- Processed dataset (anonymized)
- Analytical outputs and model summaries
- Supporting documentation for methodology

## Objective
To contribute to the broader understanding of sustainable finance behavior and decision-making mechanisms without disclosing proprietary model specifications.
